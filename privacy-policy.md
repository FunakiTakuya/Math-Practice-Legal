# 算数練習 — プライバシーポリシー

**最終更新日:** 2026年5月15日

船木 拓也（以下「当方」）は、iOS アプリ「算数練習」（以下「本アプリ」）における利用者情報の取扱いについて、以下のとおり定めます。

**本アプリは主に小学校低学年のお子さまが利用することを想定しています。保護者の方が内容をご確認のうえ、お子さまの利用をご判断ください。**

---

## 1. 適用範囲

本ポリシーは、本アプリの利用に関して当方が取得・利用する情報に適用されます。App Store 経由の課金・Apple ID に関する事項は、Apple Inc. のプライバシーポリシー等が適用されます。

---

## 2. 当方が取得する情報の種類

### 2.1 端末内に保存される情報（当方サーバーへは送信しません）

本アプリは、練習データを **お使いの端末内**（主に UserDefaults 等のローカルストレージ）に保存します。当方が運営する練習用 Web サーバーへ、これらをアップロードする機能は **現行バージョンでは提供していません**。

| 種類 | 内容の例 |
|------|----------|
| 練習セッション | 開始・終了時刻、解答の正誤、解答時間、出題数など |
| 問題ごとの統計 | 問題 ID ごとの正解・不正解回数など |
| 進捗 | 種目ごとの周回・途中位置など |
| 設定 | 入力方法、出題数、表示時間、履歴件数の設定値など |
| 途中の練習 | 中断時の下書き（再開用） |
| 無料枠の利用状況 | 当日に完了した練習の「1 周」数（日次上限の管理用） |
| Pro 状態 | サブスクリプションに基づく Pro 利用可否のフラグ（端末内） |

### 2.2 お子さまが入力する情報

- 練習中に入力した**数値（答え）**
- 音声入力モードを利用した場合、**音声データ**（下記 2.4）

これらは練習機能のために端末上で処理され、上記のとおりローカルに記録される場合があります。

### 2.3 課金に関する情報

サブスクリプション（算数練習 Pro）の購入・復元・有効期限の確認は **Apple の StoreKit** を通じて行います。当方は、**クレジットカード番号等の決済情報を直接取得しません**。取引に関する情報は Apple が管理します。

### 2.4 端末の権限を利用する場合（音声入力時）

音声入力を利用する場合、本アプリは次の権限を求めます。

| 権限 | 目的 |
|------|------|
| マイク | 音声入力のため |
| 音声認識（Speech） | 話した答えを数値として認識するため |

音声認識は **Apple が提供する OS の機能**を利用します。認識のための処理は Apple の仕様・ポリシーに従い、当方の独自サーバーへ音声を送信する設計にはしていません（OS・ネットワークの挙動は Apple のドキュメントをご参照ください）。

### 2.5 当方が意図的に収集しないもの

現行実装において、当方が以下を収集するための SDK は組み込んでいません。

- 氏名・メールアドレス・電話番号などの**アカウント登録情報**（本アプリにログイン機能はありません）
- 位置情報
- 広告 ID を用いた行動ターゲティング広告
- 第三者によるアクセス解析（Firebase Analytics 等）※将来導入する場合は本ポリシーを改定し、アプリ内でお知らせします

---

## 3. 利用目的

取得・保存した情報は、次の目的で利用します。

1. 算数練習の提供（出題、採点、結果表示、途中再開）
2. 成長グラフ・苦手問題の表示など、学習の振り返り機能の提供
3. 無料プランと Pro プランの機能制限の適用
4. サブスクリプション状態の反映（StoreKit 連携）
5. 不具合対応・お問い合わせへの対応（保護者からご連絡いただいた場合）
6. 法令に基づく対応

---

## 4. 第三者提供

当方は、次の場合を除き、利用者情報を第三者に提供しません。

- 利用者（保護者）の同意がある場合
- 法令に基づく場合
- 人の生命・身体・財産の保護に必要な場合で、同意取得が困難な場合

**Apple 社**は、App Store・StoreKit・音声認識等のサービス提供者として、それぞれのポリシーに基づき情報を処理します。

---

## 5. 保管期間・削除

- データは端末内に保存され、**アプリ内の「練習のきろくをけす」等の操作**または**アプリの削除**により消去できます。
- 無料プランでは、保存される完了セッション履歴に**件数上限**（現行: 最大 5 件）があります。古いものから保持されない場合があります。
- 当方サーバーに保存しないため、端末変更時は**自動では引き継がれません**（iCloud バックアップ等は Apple / 利用者の端末設定に依存します）。

---

## 6. お子さまの利用について

本アプリは教育目的のツールです。13 歳未満のお子さまに利用させる場合は、**保護者の方が利用内容・本ポリシー・利用規約を確認し、同意のうえでご利用ください**。課金操作は保護者の方の管理下で行うことを想定しています。

---

## 7. 安全管理

端末内保存を前提とし、不要な外部送信を行わない設計としています。お問い合わせで個人情報をいただく場合は、目的達成に必要な範囲で適切に管理します。

---
