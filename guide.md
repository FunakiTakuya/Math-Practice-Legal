# 算数練習 — 使い方ガイド

小学校低学年向けの計算ドリルアプリです。端末のホーム画面では **算数練習**、App Store では **毎日 計算れんしゅう** と表示されます。アプリ内では **さんすう れんしゅう** と表示されます。

---

## アプリを入れる

**ストアで開く:** [毎日 計算れんしゅう（App Store）](https://apps.apple.com/app/毎日-計算れんしゅう/id6775496517)

  <img src="images/app-store-qr.png" alt="App Store への QR コード" width="160" style="max-width: 160px; width: 160px; height: auto;" />

1. iPhone / iPad のカメラで QR を写す → 表示されたリンクをタップ  
2. App Store で **入手**（または **再ダウンロード**）をタップ

---

## 練習の基本の流れ

1. アプリを開く → **ホーム**
2. 練習の種類を選ぶ（例: **たしざん・ひきざん**）
3. 種目を選ぶ（例: **たしざん（1）**、**10をつくる**）
4. 問題に答える → **OK**
5. 全問終わると **練習完了**（記録が残る）

**途中でやめるとき**

| 操作 | 結果 |
|------|------|
| **やめる** → **いちじちゅうだん** | 途中まで保存。ホームの **つづきから** で再開 |
| **やめる** → **ほんとうに やめる** | その回は保存されない |

---

## ホーム画面

| 表示 | 意味 |
|------|------|
| たしざん・ひきざん など | 練習の種類を選ぶ |
| **れんぞく ○ にち** | 続けて練習した日数 |
| **つづきから** | 中断した練習を続ける（あるときだけ表示） |
| **せってい** | 入力のしかた・ヒント・保護者向けメニュー |

---

## 種目の選び方

### 無料で選べる種目

| 種目 | どんな練習か |
|------|----------------|
| たしざん（1） | 答えが 10 までの足し算 |
| ひきざん（1） | 答えが 10 までの引き算 |
| **10をつくる** | 例: `3 + ? = 10`（10 になる数を答える） |
| **10を使う** | 例: `10 − 7`（10 から引く） |

鍵マークの種目は **算数練習 Pro** が必要です（繰り上がり・繰り下がり、かけざん・わりざんなど）。

---

## 練習中の操作

### 答え方（**せってい**で切り替え）

| 入力 | 使い方 |
|------|--------|
| **テンキー** | 数字を押して **OK** |
| **音声** | こたえを声に出す（はじめて使うときマイクの許可が必要） |
| **未入力** | 答えを入れず、正解だけ確認する |

音声がうまくいかないときは、練習画面から **テンキー** に切り替えられます。

### ヒント

- 出題画面の **ヒント** で、計算のしかたを見られます
- **せってい** → **ヒントボタン** で表示のオン/オフ
- ヒントを開いているあいだは答えられません。**とじる** で戻ります

### 練習が終わったあと

- **きょう ○ かいめ** … その日に何回目の完了か
- 速さの記録が出ることがあります

---

## 成長の記録

**せいちょうの きろく**（ホームまたは種目の画面から）

- 種目ごとの **グラフ** で練習の推移を見る
- **にがて** … よく間違える問題の傾向

---

## 設定（せってい）

### お子さまがよく使う項目

- **入力方法** … テンキー / 音声 / 未入力
- **ヒントボタン** … 出題画面にヒントを出すか
- **タイム表示** … 練習中・結果の時間（タイマー）を出すか。時間が気になるときは「ださない」に
- **れんしゅうのじかん** … 毎日のおしらせ（リマインダー）

**れんしゅうのじかん（毎日のおしらせ）**

1. 「**する**」を選ぶ → おしらせの許可 → 時刻のダイヤルで時間を選ぶ → 「**けってい**」
2. 設定すると「する」が「**かえる**」に変わり、いつでも時刻を変えられます
3. 「**しない**」でおしらせを止められます（時刻は覚えているので、また「する」で戻せます）

※ おしらせは端末の中だけで動きます。時刻は朝から夜まで自由に選べます。

### おうちの方へ（保護者向け）

| メニュー | できること |
|----------|------------|
| **練習のまとめ** | 直近7日間の練習回数・正答率など。日付をタップするとその日の詳細 |
| **Pro について** | Pro の説明・購入・**購入を復元する** |

**練習のまとめ（無料）** … 直近7日間のストリップ。練習した日をタップで詳細。

**練習のまとめ（Pro）** … 月カレンダー。左スワイプで前の月。

---

## 無料で使うときの上限

| 内容 | 無料 |
|------|------|
| 1日に完了できる練習 | **3 回まで**（1回＝その種目を一通り終える） |
| 残る練習履歴 | **5 件まで** |

3 回に達すると **きょうの じょうげん** と表示され、その日は新しい練習を完了して保存できません。Pro ではこの上限がありません。

---

## 困ったとき

| 状況 | 対処 |
|------|------|
| 音声が聞き取られない | 練習画面でテンキーに切り替え、または **せってい** → 入力方法をテンキーに |
| 記録を消したい | **せってい** 内のリセット（履歴・統計・進捗を削除） |
| Pro を買ったのに使えない | **おうちの方へ** → **Pro について** → **購入を復元する** |
