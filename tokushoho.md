## 所在地・電話番号（請求により開示）

当方は **通信販売** に該当する提供のため、特定商取引法上、**所在地および電話番号は、お客様からの請求があった場合に、遅滞なく開示**します（常時の掲載は行いません）。

## 代金の支払時期・方法

- **支払時期:** 購入確定時に Apple ID に紐づく支払方法へ請求。以降、契約期間満了前の自動更新時に課金されます。
- **支払方法:** Apple が提供する App Store 経由の決済（クレジットカード、キャリア決済、Apple ID 残高等、利用者の設定による）。

- 購入・復元が完了した時点から、Pro 機能を利用できます（通信環境・端末・Apple のサービス状況による）。

## 返品・キャンセル・解約

| 項目 | 内容 |
|------|------|
| **解約** | iPhone の **設定 → Apple ID → サブスクリプション** から、期間終了前に解約できます。アプリ内の「購入を復元する」で以前の購入を復元できる場合があります。 |
| **自動更新** | 解約しない限り契約は更新されます。更新の **24 時間前まで** に解約しないと、次期間分が課金されます。 |
| **返金** | 原則として Apple の返金ポリシーに従います。[reportaproblem.apple.com](https://reportaproblem.apple.com/) 等、Apple 経由でお申し込みください。当方から直接返金するものではありません。 |
| **デジタルコンテンツ** | ダウンロード済みのデジタル役務の性質上、提供開始後の返品・返金はお受けできない場合があります（Apple 経由の返金手続を除く）。 |
